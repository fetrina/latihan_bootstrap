Navigation Bar (basic)

Navigation bar dihasilkan dgn class="navbar" pd tag nav. Setelah tag nav (pembuka), diberikan div class="container-fluid", agar lebar memenuhi seluruh layar.

Didalam container, kt buat div class="navbar-header" yg membungkus elemen button, juga a href.

Brand

NavBar Brand

Brand merupakan sebuah link dasar yg memperlihatkan merk atau nama proyek atau logo perusahaan.

Tag nav class="navbar navbar-default", untuk membuat navigation bar.

div class="navbar-header", untuk membungkus tag a href yg memakai class="navbar-brand".

Posisi Pager

Tag ul dengan class="pager", untuk membuat dua buah link pagination. Dua buah link kita buat dlm tag li dan a href.

Posisi diubah rapat ke kiri dengan class="previous" pada tag li. Untuk rapat ke kanan, berikan class="next".
